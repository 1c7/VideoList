#### 3. 这些视频有完整的正确英文字幕吗?
没有.. __只有英文自动字幕..__   

什么是英文自动字幕?   
因为Youtube有语音识别技术  所以可以用机器把英文视频的英文字幕做出来   
正确率大概在70%~95%之间 已经相当不错了   
(目前只支持英文)
